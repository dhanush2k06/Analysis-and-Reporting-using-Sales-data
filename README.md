# 🛒 E-Commerce Sales Analysis Dashboard

## 📌 Project Overview

This project presents a comprehensive **E-Commerce Sales Analysis** using **Microsoft Excel** for data analysis and **Power BI** for dashboard visualization. The objective is to analyze sales performance, customer behavior, profitability, and operational efficiency to support data-driven business decisions.

The project simulates a real-world business case study where stakeholders seek insights into revenue generation, customer satisfaction, pricing strategy, and logistics performance.

---

## 🎯 Business Objectives

The analysis answers the following business questions:

1. Which Top 10 products generate the highest revenue?
2. Which product categories generate the highest profit?
3. How do monthly sales vary throughout the year?
4. Which regions generate the highest sales?
5. How do discounts impact sales and profitability?
6. Who are the top customers by revenue?
7. Which payment method is preferred by customers?
8. Which product categories receive the lowest customer ratings?
9. Which regions experience the longest average shipping time?
10. Which product categories have the highest return rates?

---

## 📊 Dataset Information

A custom dataset containing **200 e-commerce transactions** was created for this project.

### Dataset Features

| Column | Description |
|----------|-------------|
| OrderID | Unique identifier for each order |
| OrderDate | Date of purchase |
| CustomerID | Unique customer identifier |
| Region | Customer location |
| Category | Product category |
| Product | Product name |
| Quantity | Number of units purchased |
| UnitCost | Cost per unit |
| UnitPrice | Selling price per unit |
| DiscountPercent | Discount applied (%) |
| Sales | Total sales amount |
| Profit | Profit earned |
| PaymentMethod | Mode of payment |
| Rating | Customer rating (1–5) |
| ShippingDays | Delivery time in days |
| Returned | Indicates whether the product was returned |
